title: 博客的创建
date: 2025-2-1 17:49:32
tags: blog
----------

1.在终端下使用

```
cnpm install hexo-cli -g
```

或

```
npm install hexo-cli -g
```

2.创建文件夹

```
hexo init blog
```

3,进入blog

```
cd blog
```

4.npm install 安装依赖

```
npm install 
```

5.启动博客

```
hexo server
```

或

```
hexo server -p 4000
```
